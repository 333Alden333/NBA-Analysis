Create four SQLAlchemy model files following the established `Mapped`/`mapped_column` pattern from existing models. All inherit from Base.

**PlayerRollingStats** (`player_rolling_stats`):
- `id` (int, PK), `player_id` (int, FK players.player_id, indexed), `game_id` (str(10), FK games.game_id, indexed), `game_date` (Date, indexed)
- For each of 12 stats (points, rebounds, assists, steals, blocks, turnovers, fg_pct, fg3_pct, ft_pct, minutes, plus_minus, offensive_rebounds) and each of 3 windows (5, 10, 20): `{stat}_avg_{window}` (Float, nullable). That is 36 rolling average columns.
  - Note: For percentage stats (fg_pct, fg3_pct, ft_pct), compute as rolling mean of per-game percentages. Name them `fg_pct_avg_5`, etc.
- `games_available_5`, `games_available_10`, `games_available_20` (Integer, nullable) -- how many games were in the window
- UniqueConstraint('player_id', 'game_id', name='uq_player_game_rolling')

**PlayerAdvancedStats** (`player_advanced_stats`):
- `id` (int, PK), `player_id` (int, FK, indexed), `game_id` (str(10), FK, indexed), `game_date` (Date, indexed)
- `true_shooting_pct` (Float, nullable), `usage_rate` (Float, nullable), `simplified_per` (Float, nullable)
- `team_fga`, `team_fta`, `team_tov` (Integer, nullable) -- stored for auditability
- `team_minutes` (Float, nullable)
- UniqueConstraint('player_id', 'game_id', name='uq_player_game_advanced')

**MatchupStats** (`matchup_stats`):
- `id` (int, PK), `player_id` (int, FK, indexed), `game_id` (str(10), FK, indexed), `game_date` (Date, indexed)
- `opponent_team_id` (int, FK teams.team_id, indexed)
- `matchup_games_played` (Integer) -- number of historical meetings used
- `has_matchup_history` (Boolean, default=False) -- True if >= 3 games of history
- For key stats (points, rebounds, assists, fg_pct, plus_minus): `matchup_avg_{stat}` (Float, nullable) and `matchup_diff_{stat}` (Float, nullable, difference from player's overall average)
- UniqueConstraint('player_id', 'game_id', name='uq_player_game_matchup')

**TeamFeatures** (`team_features`):
- `id` (int, PK), `team_id` (int, FK teams.team_id, indexed), `game_id` (str(10), FK, indexed), `game_date` (Date, indexed)
- `pace` (Float, nullable), `offensive_rating` (Float, nullable), `defensive_rating` (Float, nullable)
- `rest_days` (Integer, nullable)
- `possessions` (Float, nullable), `opponent_possessions` (Float, nullable)
- `season_win_pct` (Float, nullable) -- rolling win percentage up to this game
- UniqueConstraint('team_id', 'game_id', name='uq_team_game_features')
