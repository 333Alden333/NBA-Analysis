    - test_defensive_rating: DRtg = (OppPoints / Possessions) * 100. Opponent scores 105 on 100 possessions -> DRtg = 105.0
    - test_rest_days: Team plays Jan 1, Jan 3, Jan 6. Rest for Jan 3 game = 2, rest for Jan 6 game = 3. First game of season = 3.
    - test_season_win_pct: After 10 games with 7 wins, season_win_pct for next game = 0.7

  <action>
Create `src/hermes/data/features/team.py` with:

**Pure formula functions** (no DB dependency):
- `estimate_possessions(fga, fta, orb, tov) -> float` -- `FGA + 0.44*FTA - ORB + TOV`. If ORB is None/unavailable, use simplified: `FGA + 0.44*FTA + TOV`.
- `compute_offensive_rating(points, possessions) -> float` -- `(points / possessions) * 100`. Returns 0.0 if possessions == 0.
- `compute_pace(team_poss, opp_poss, team_minutes) -> float` -- `((team_poss + opp_poss) / 2) * (240 / team_minutes)`. Returns 0.0 if team_minutes == 0.
- `compute_rest_days(current_game_date, previous_game_date) -> int` -- `(current - previous).days`. Returns 3 if previous is None (season opener).

**DB-integrated function:**
- `compute_team_features(session, team_id, as_of_date=None)`:
  1. Query all games for this team (as home or away), ordered by game_date ASC. If as_of_date, filter `game_date < as_of_date` for rolling features BUT still compute features FOR the current game using only prior data.
  2. For each game: aggregate BoxScore stats for team and opponent to get team_fga, team_fta, team_orb, team_tov, team_points, team_minutes AND opponent equivalents.
  3. Compute possessions for team and opponent using `estimate_possessions()`.
  4. Compute pace, ORtg, DRtg for this game.
  5. Compute rest_days from gap between current game and previous game for this team.
  6. Compute season_win_pct: count of team's wins in games before this game (team was home and home_score > away_score, or team was away and away_score > home_score) divided by total games played before this game.
  7. Upsert to TeamFeatures table.

- `compute_team_features_for_games(session, game_ids)`:
  Convenience: find all team_ids in those games, call compute_team_features for each.

Note on open question about offensive_rebounds: Use `estimate_possessions` with ORB when available (offensive_rebounds column exists in BoxScore). If SUM(offensive_rebounds) is NULL for a game, fall back to simplified formula without ORB.

Update test stubs with real test implementations.
